# Layer 1 Physical — Control Map

| NIST Control | Control Name | Tool | Enterprise Equivalent | What Misconfiguration Looks Like |
|-------------|-------------|------|----------------------|--------------------------------|
| PE-1 | Physical Security Policy | Documented policy | GRC platform (Archer, ServiceNow) | No written policy, no annual review, no enforcement mechanism |
| PE-2 | Physical Access Authorizations | Access control list | Identity governance (SailPoint) | Stale access list, terminated employees still authorized |
| PE-3 | Physical Access Control | Badge readers, mantraps | Enterprise PACS (Lenel, AMAG) | Tailgating, propped doors, no visitor escort policy |
| PE-6 | Monitoring Physical Access | CCTV, access logs | Enterprise VMS (Genetec, Milestone) | Cameras with blind spots, logs not reviewed, no alerting |
| PE-13 | Fire Protection | Suppression systems | FM-200, Novec 1230 | No suppression in server room, no smoke detection, no evacuation plan |
| PE-14 | Environmental Controls | HVAC sensors, alerting | BMS (Honeywell, Siemens) | No temperature monitoring, no redundant HVAC, no auto-shutdown threshold |
