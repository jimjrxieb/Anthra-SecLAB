# Incident Response — Control Map

| NIST Control | Control Name | Tool | Enterprise Equivalent | What a Gap Looks Like |
|-------------|-------------|------|----------------------|----------------------|
| IR-4 | Incident Handling | TheHive, IRIS | ServiceNow ITSM, IBM Resilient | No documented IR procedure, no case management, improvised response every time |
| IR-5 | Incident Monitoring | TheHive, Splunk | Splunk SIEM, Microsoft Sentinel | Incidents not tracked or documented, no post-incident metrics |
| IR-6 | Incident Reporting | Ticketing system, TheHive | ServiceNow, Jira | No defined escalation path, no regulatory reporting procedure, incidents go unreported |
| IR-8 | Incident Response Plan | Documentation, tabletop exercises | Tabletop exercises, IR retainers (CrowdStrike, Mandiant) | No written IR plan, no tested procedures, no contact list for escalation |
