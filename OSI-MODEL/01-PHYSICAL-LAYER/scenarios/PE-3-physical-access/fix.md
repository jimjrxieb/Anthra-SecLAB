# PE-3 Physical Access Control — Fix

## Remediation Steps

### Immediate (0-30 days)

1. **Anti-tailgating policy** — publish written policy requiring one-person-per-badge-swipe
2. **Security awareness training** — brief all employees on tailgating risks and challenge procedures
3. **Signage** — post "One badge, one entry" signs at all controlled access points
4. **Visitor escort policy** — require badge-holder escort for all visitors, no exceptions

### Short-term (30-90 days)

1. **Anti-tailgating hardware** — install turnstiles or mantrap vestibules at high-security entry points (data center, server rooms)
2. **Occupancy sensors** — deploy sensors that count entries and alert when count exceeds badge swipes
3. **CCTV upgrade** — ensure cameras cover all badge-controlled entry points with clear facial capture

### Long-term (90+ days)

1. **Biometric second factor** — add fingerprint or iris scan as second factor at data center entry
2. **Automated challenge** — integrate badge system with CCTV for real-time alerting on unbadged entry
3. **Regular penetration testing** — include physical social engineering in annual pen test scope
